# Проект: Аномалия / Люденс / Игра  
_Рабочее название, потом можно переименовать_

## О чем этот проект

Это зачаток романа / цикла в сеттинге, где:

- На Земле происходит **аномальный шторм и цунами**.  
- В центре катастрофы появляется **защитное поле**: город в кольце воды, как в аквариуме.  
- Власти создают карантин, эвакуируют население, запускают секретное расследование.  
- В процессе выясняется, что:
  - на Земле действуют **«чужие» силы** с технологиями сильно выше человеческих;
  - эти силы конфликтуют между собой;
  - сама Земля — часть **Игры**, управляемой цивилизацией **Люденс** и **Комитетом Дизайнеров (КД)**;
  - инцидент со штормом и защитным полем — **сбой / вмешательство** на уровне фундаментальных протоколов реальности.

История разворачивается на двух уровнях:

1. **Уровень Земли** — военные, спецслужбы, ученые, политики; катастрофа, аномалия, контакт с «чужими», вопрос суверенитета и безопасности.
2. **Уровень Люденс / Администрации Игры** — расследование инцидента как нарушения протоколов Игры, конфликт Владельцев, Комитета, Игроков и пре-Люденс, попытки удержать сюжет мира в контролируемом коридоре.

Планируемый объем:  
- Часть 1 — ~100 000 слов (Земля, аномалия, контакт, спецоперации).  
- Часть 2 — ~100 000 слов (Администрация Игры, Люденс, Комитет, метафизика, реконструкция инцидента).

## Что уже есть в этом репозитории (по замыслу)

Текущий текст — это смесь:

- **Событийных блоков** (Событие 1–7) — ключевые эпизоды:
  - шторм и возникновение поля;
  - эвакуация, карантин, расследование;
  - первые гипотезы (инопланетное / природное / земная технология);
  - выявление «чужих» сил;
  - предложение о сотрудничестве от «хороших парней» против «плохих»;
  - первая встреча и переговоры;
  - внутренний разбор на стороне землян.

- **Лор / мета-уровень**:
  - Игровые миры, Владельцы, Комитет Дизайнеров;
  - Люденс, пре-Люденс, Игроки, NPC;
  - протоколы реальности, «разумные сюжеты», Стена Рекурсии;
  - конфликты между Владельцами, Комитетом, Игроками;
  - теория о том, почему Человек и ИИ принципиально различаются на уровне “инсайта”.

- **План и вопросы**:
  - структура части 1 и 2;
  - роли следователей Земли / Владельцев / КД;
  - ключевые проблемы (кто создал шторм, кто поставил защиту, что защищалось, как это связано с архитектурой Игры);
  - направления для персонажей, сценок и диалогов.

## Цели репозитория

1. **Собрать команду авторов**, работающих над:
   - черновыми сценами,
   - диалогами,
   - персонажами,
   - проработкой лора.

2. **Держать в порядке канон**:
   - события,
   - причинно-следственные связи,
   - устройство мира,
   - терминологию.

3. **Использовать GitHub как “литературный Git”**:
   - есть одна **главная версия текста**;
   - каждый автор работает в своей **личной копии**;
   - изменения попадают в главный текст только через **предложения изменений** (pull request).
