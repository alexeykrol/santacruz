# Акт 1: Инцидент

**Временной охват:** Параллельно с Частью 1

## Описание

События глазами Администрации Игры. Обнаружение грубого вмешательства в операционные протоколы физики мира. Формирование группы КД для расследования. Решение о режиме "видимости" для землян.

## Ключевые сцены (примерный план)

1. **Алерт системы мониторинга** - обнаружение аномалии
2. **Анализ вмешательства** - кто-то перехватил управление физикой
3. **Совещание Владельцев** - что делать с Игрой?
4. **Комитет vs Владельцы** - конфликт интересов
5. **Формирование группы КД** - выбор оперативников
6. **Решение о "видимости"** - позволить землянам видеть следы
7. **Параллельное расследование** - КД ищут то же, что земляне

## Вводимые концепции

- Протоколы реальности
- Разумные Сюжеты (РС)
- Владельцы vs Комитет Дизайнеров
- Игроки и пре-Люденс
- Этический кодекс

## Тон и атмосфера

- Деловая эффективность
- Высокий профессионализм
- Обеспокоенность серьезностью инцидента
- Бюрократия мета-уровня
