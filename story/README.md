# /story - Текст произведения

Эта папка содержит сам текст романа, организованный как последовательность **сцен**.

## Принцип организации

Сцена - это основная единица повествования, контейнер со всеми необходимыми элементами:
- Временная метка
- Локация и точка пространства
- Субъекты действия (персонажи)
- Объекты
- Действие/Диалог

## Структура

```
/part1_earth/        - Часть 1: Сеттинг Земли (~100,000 слов)
/part2_admin/        - Часть 2: Сеттинг Люденс (~100,000 слов)
```

## Как читать

Текст читается **строго последовательно**, сцена за сценой, как HTML-код в браузере.
