# /drafts/master - Книжная версия

Эта папка содержит **единый текст романа** в книжном формате.

## Назначение

Весь текст из эпизодов собирается в единое повествование, организованное как книга:
- Разбито по частям (Часть 1, Часть 2)
- Разбито по главам
- Разбито по эпизодам
- Непрерывное чтение

## Отличие от /drafts/episodeXXX/

| Критерий | /episodeXXX/ | /master/ |
|----------|--------------|----------|
| **Назначение** | Публикация в рассылке | Книжный формат |
| **Структура** | Отдельные эпизоды | Единый текст |
| **Формат** | 1 эпизод = 10-12k символов | Последовательное повествование |
| **Навигация** | По эпизодам | По частям → главам → эпизодам |

## Структура

```
/master/
├── README.md
├── /part1_earth/
│   ├── chapter01_storm.md         # Глава 1 (может содержать несколько эпизодов)
│   ├── chapter02_investigation.md # Глава 2
│   └── chapter03_contact.md       # Глава 3
│
└── /part2_admin/
    ├── chapter01_incident.md
    ├── chapter02_search.md
    └── chapter03_resolution.md
```

## Формат файла главы

Каждая глава - это Markdown файл с метаданными:

```markdown
---
chapter: 1
part: 1
title: "Шторм"
episodes: [1, 2, 3]
word_count: ~35000
---

# Глава 1. Шторм

## Эпизод 1: Начало

[Текст эпизода 1...]

---

## Эпизод 2: Поле

[Текст эпизода 2...]

---

## Эпизод 3: Эвакуация

[Текст эпизода 3...]
```

## Процесс сборки

### Автоматическая сборка (идеально)
Скрипт автоматически собирает эпизоды в главы:
```bash
./scripts/build_master.sh
```

### Ручная сборка
1. Копируете текст из `/drafts/episodeXXX/`
2. Вставляете в соответствующую главу в `/master/`
3. Форматируете (добавляете разделители между эпизодами)

## Использование

### Для читателей
Master-версия позволяет читать роман как цельное произведение, а не отдельные эпизоды.

### Для издания
Master-версия - это то, что пойдет в печать/на публикацию как книга.

### Для навигации
Быстро понять структуру всего романа и найти нужный фрагмент.

## Форматирование

### Заголовки
- `# Глава N. Название` - заголовок главы
- `## Эпизод N: Название` - заголовок эпизода
- `### Подзаголовок` - если нужен внутри эпизода

### Разделители
Между эпизодами:
```markdown
---
```

Между сценами внутри эпизода (опционально):
```markdown
* * *
```
