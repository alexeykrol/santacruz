# /meta - Метаинформация

Эта папка содержит **метаинформацию о проекте** - заметки, планы, вопросы, которые не являются частью мира или текста.

## Структура

### [/notes/](./notes/) - Заметки
- **⭐ ontology_to_story.md** - Маппинг онтологии на элементы сюжета Санта-Круз (референсы, таблицы связей)
- **questions.md** - Открытые вопросы и противоречия, требующие решения
- **conflicts.md** - Конфликты для проработки
- **ideas.md** - Идеи для сцен, персонажей, сюжетных линий

### [/plans/](./plans/) - Планы
- **part1_outline.md** - План Части 1
- **part2_outline.md** - План Части 2
- **scenes_roadmap.md** - Дорожная карта сцен

### [scene_ideas.md](./scene_ideas.md)
Идеи для сцен, извлеченные из development.md и других источников.

### [glossary.md](./glossary.md)
Глоссарий терминов проекта. Краткие определения всех специальных терминов.

## Назначение

Meta - это **рабочее пространство** для авторов:
- Фиксировать идеи
- Планировать развитие
- Отслеживать проблемы
- Координировать работу

## Отличие от lore

- **Lore** - информация о мире (для читателей и авторов)
- **Meta** - информация о проекте (только для авторов)

## Примеры содержимого

**questions.md:**
> - Как КД проникли в ядро управления физикой?
> - Кто именно создал шторм и зачем?
> - Что было в эпицентре, что требовало защиты?

**ideas.md:**
> - Сцена: Первая ночь после аномалии, местный житель не может уснуть, смотрит на светящийся купол
> - Персонаж: Журналист, который пытается разоблачить заговор
