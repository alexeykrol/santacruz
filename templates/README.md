# /templates - Шаблоны

Эта папка содержит **шаблоны** для создания новых файлов.

## Доступные шаблоны

### [episode_readme_template.md](./episode_readme_template.md)
Шаблон README для эпизода (метаданные эпизода для рассылки).

### [scene_template.md](./scene_template.md)
Шаблон для написания сцены со всеми необходимыми метаданными.

### [character_template.md](./character_template.md)
Шаблон для описания персонажа.

### [location_template.md](./location_template.md)
Шаблон для описания локации.

## Как использовать

1. Скопируйте нужный шаблон
2. Заполните все секции
3. Сохраните в соответствующую папку

## Зачем нужны шаблоны

Шаблоны обеспечивают:
- **Единообразие** - все файлы одного типа имеют одинаковую структуру
- **Полноту** - не забудете важные детали
- **Консистентность** - легко поддерживать стиль проекта
