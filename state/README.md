# /state - Состояние (переменные)

Эта папка содержит **изменяемое состояние** - переменные, которые меняются по ходу сюжета.

## Отличие от констант

- **Константы** (в `/worlds/`, `/settings/`) - не меняются
- **Переменные** (здесь) - изменяются по ходу повествования

## Файлы состояния

### [timeline.md](./timeline.md)
Хронология событий. Временные метки всех ключевых событий и сцен.

### [character_states.md](./character_states.md)
Состояния персонажей по фазам сюжета:
- Физическое состояние
- Эмоциональное состояние
- Уровень знаний
- Отношения

### [investigation_progress.md](./investigation_progress.md)
Прогресс расследования:
- Список артефактов (16,000 → 79 → 50)
- Список подозреваемых (143 → 100 → ?)
- Обнаруженные факты
- Текущие гипотезы

### [relationships.md](./relationships.md)
Отношения между персонажами и группами:
- Земляне ↔ КД (от подозрения к сотрудничеству)
- КД ↔ Владельцы
- Владельцы ↔ Комитет

### [threat_level.md](./threat_level.md)
Уровень тревоги и статусы:
- Уровень тревоги (синий, желтый, красный)
- Статус карантина
- Статус расследования

## Как использовать

При написании сцены:
1. Проверьте текущее состояние персонажей
2. Проверьте прогресс расследования
3. Обновите состояние после событий сцены

## Пример

**До сцены:**
- Артефактов под наблюдением: 79
- Уровень тревоги: обычный

**События сцены:**
- Обнаружены следы "чужих"

**После сцены:**
- Артефактов под наблюдением: 79 (не изменилось)
- Уровень тревоги: синий (изменилось)
