# /settings - Сеттинги

Эта папка содержит описание **сеттингов** - историко-культурных контекстов, в которых разворачивается действие.

## Что такое "Сеттинг"

Сеттинг - это более конкретное понятие, чем Мир. Это комбинация:
- **Исторической эпохи** (Древний Египет, Средневековье, 2025 год)
- **Географического региона** (Европа, Азия, Америка)
- **Культурного контекста** (общество, институты, нормы)

## Отличие от Мира и Локации

- **Мир** = физические и социальные константы (гравитация, нельзя сквозь стену)
- **Сеттинг** = эпоха + культура + география (современная США, 2025)
- **Локация** = конкретное место (Нью-Йорк, Санта-Круз)

## Что содержит описание сеттинга

- **overview.md** - общее описание эпохи и контекста
- **institutions.md** - институты и их правила (военные, спецслужбы, полиция)
- **rules.md** - специфические правила, действующие в этом контексте
