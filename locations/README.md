# /locations - Локации

Эта папка содержит описание **локаций** - географических мест, где разворачивается действие.

## Что такое "Локация"

Локация - это **широкое географическое понятие**: город, регион, область.

Примеры:
- Нью-Йорк
- Санта-Круз
- Сан-Франциско
- Калифорния
- Карантинная зона

## Отличие от Точки пространства

- **Локация** = широкое место (город, регион)
- **Точка пространства** = конкретное место (улица, здание, комната)

Пример:
- Локация: Нью-Йорк
- Точка пространства: Конференц-зал на Манхэттене, 42-я улица, 15 этаж

## Структура

```
/earth/          - Локации на Земле
/ludens/         - Локации в мире Люденс
```

## Что содержит описание локации

- Географическое положение
- Общая характеристика
- Значение для сюжета
- Ключевые точки пространства внутри локации
