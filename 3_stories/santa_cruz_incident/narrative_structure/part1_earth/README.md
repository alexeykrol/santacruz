# /story/part1_earth - Часть 1: Сеттинг Земли

**Объем:** ~100,000 слов
**Фокус:** События на уровне Земли - военные, спецслужбы, ученые, политики

## Описание

Катастрофа, аномалия, контакт с "чужими", вопрос суверенитета и безопасности.

## Структура актов

### Акт 1: Аномалия ([act1_anomaly/](./act1_anomaly/))
- Шторм и цунами
- Возникновение защитного поля
- Первые реакции и наблюдения

### Акт 2: Расследование ([act2_investigation/](./act2_investigation/))
- Эвакуация и карантин
- Формирование следственной группы
- Поиск артефактов и подозреваемых
- Обнаружение "чужих"

### Акт 3: Контакт ([act3_contact/](./act3_contact/))
- Предложение от "хороших парней"
- Первая встреча в Нью-Йорке
- Начало сотрудничества
- Внутренний разбор у землян

## Ключевые локации

- Санта-Круз (зона аномалии)
- Сан-Франциско (операционный центр)
- Нью-Йорк (место встречи)
