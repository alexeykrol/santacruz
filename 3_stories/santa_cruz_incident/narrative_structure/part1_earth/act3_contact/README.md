# Акт 3: Контакт

**События:** 5-7 из development.md
**Временной охват:** Несколько недель

## Описание

Расследование заходит в тупик. Внезапно группа получает странное предложение от "хороших парней" - инопланетян, охотящихся за "плохими парнями". Первая встреча в Нью-Йорке. Демонстрация силы. Начало сотрудничества.

## Ключевые сцены (примерный план)

1. **Тупик** - следствие не двигается
2. **Странное предложение** - сообщение с сервера
3. **Взлом защищенных центров** - демонстрация возможностей
4. **Подготовка к встрече** - формирование стратегического штаба
5. **Встреча на Манхэттене** - конференц-зал
6. **Отключение жучков** - демонстрация контроля
7. **Легенда "хороших парней"** - объяснение ситуации
8. **"Мы можем остановить планету"** - угроза без надменности
9. **Исчезновение из лифта** - финальная демонстрация
10. **Совещание штаба** - внутренний разбор у землян

## Тон и атмосфера

- От отчаяния к надежде
- Унижение от технологического разрыва
- Осознание заложничества в ситуации
- Прагматизм землян
