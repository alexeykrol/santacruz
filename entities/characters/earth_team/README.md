# Земная следственная группа

Межведомственная группа, расследующая аномалию в Санта-Круз.

## Состав

### Руководство
- **lead_investigator.md** - Главный следователь (координация расследования)
- **military_commander.md** - Военный командир (безопасность, оперативные действия)
- **scientist_chief.md** - Главный ученый (анализ аномалии)

### Оперативники
- **intelligence_officer.md** - Офицер разведки (сбор информации о "чужих")
- **field_agent_01.md** - Полевой агент (наблюдение, поиск)

### Аналитики
- Аналитики данных, профайлеры, специалисты

## Задачи группы

1. **Фаза 1 (События 1-3):** Понять природу аномалии
2. **Фаза 2 (Событие 3-4):** Найти артефакты и подозреваемых
3. **Фаза 3 (Событие 4-5):** Выявить "чужих"
4. **Фаза 4 (События 5-7):** Сотрудничество с КД

## Полномочия

- Прямой доклад руководству страны
- Неограниченные ресурсы
- Межведомственная координация
- Максимальная секретность

## Конфликт

Группа находится в положении **заложников ситуации**:
- Нет технологий для противостояния "чужим"
- Нет рычагов влияния
- Вынуждена сотрудничать на условиях КД
