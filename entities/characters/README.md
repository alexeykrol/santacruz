# /entities/characters - Персонажи

Персонажи - это **субъекты действия**, те, кто совершает действия и участвует в диалогах.

## Структура

```
/earth_team/              - Земная следственная группа
/committee_designers/     - Представители КД (под видом "пришельцев")
/suspects/                - Подозреваемые в расследовании
/civilians/               - Гражданские лица (свидетели, статисты)
```

## Шаблон описания персонажа

Каждый персонаж описывается по единому шаблону:

```markdown
# Имя персонажа

## Базовая информация
- **Роль:** Главный следователь / Оперативник КД / и т.д.
- **Возраст:**
- **Происхождение:** Земля / Люденс

## Внешность
- Описание внешности

## Характер
- Ключевые черты характера

## Навыки и способности
- Что умеет делать

## Мотивация
- Что им движет

## Арка персонажа
- Как меняется по ходу сюжета

## Отношения
- С кем и как взаимодействует

## Первое появление
- Ссылка на сцену
```

## Важно

- Описание персонажа = справочник для авторов
- Реальное раскрытие персонажа происходит в сценах
- Поддерживайте консистентность
