# /entities - Сущности

Эта папка содержит описание **сущностей** - "классов" и "объектов" мира.

## Типы сущностей

### [/characters/](./characters/) - Персонажи
Субъекты действия. Те, кто совершает действия и участвует в диалогах.

### [/organizations/](./organizations/) - Организации
Группы, институты, фракции. Имеют свои цели, правила, структуру.

### [/objects/](./objects/) - Объекты
Важные для сюжета предметы, артефакты, технологии.

## Философия описания

Сущности описываются как **определения классов в ООП**:
- Свойства (характеристики)
- Возможности (что может делать)
- Ограничения (что не может)
- Отношения с другими сущностями

## Важно

Описания сущностей - это **справочник**, а не текст романа. Это база данных для авторов, чтобы поддерживать консистентность.

## Пример

Вместо текста романа:
> "Джон вошел в комнату, его серые глаза сканировали пространство..."

В описании персонажа:
```markdown
# Джон Смит

**Роль:** Главный следователь
**Возраст:** 45
**Внешность:** Серые глаза, седеющие виски
**Навыки:** Дедукция, психология, переговоры
```
