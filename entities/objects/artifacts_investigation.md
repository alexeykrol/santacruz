# Артефакты зоны - Расследование и классификация

## Определение

**Артефакты зоны** - объекты, обнаруженные в эпицентре аномалии (Санта-Круз), потенциально связанные с созданием защитного поля или других аномальных явлений.

**Критерий включения:**
Всё странное заносится в базу данных.

**Результат:**

## Типология артефактов

### Категория A: Технические устройства

**Признаки:**
- Неизвестная технология
- Функция не ясна
- Материалы вне земного уровня (?)

**Примеры (гипотетические):**
- Компактные устройства неизвестного назначения
- Объекты с необычными электромагнитными свойствами
- Предметы, которые не сканируются стандартными приборами

### Категория B: Аномальные объекты

**Признаки:**
- Необъяснимые физические свойства
- Нарушают известные законы физики (?)
- Связаны с эпицентром аномалии

**Примеры (гипотетические):**
- Объекты с переменным весом
- Предметы, температура которых не меняется
- Объекты, которые невозможно разрушить

### Категория C: Информационные носители

**Признаки:**
- Содержат информацию
- Возможно, данные о создателях аномалии
- Записи, логи, следы активности

**Примеры (гипотетические):**
- Странные записи на устройствах
- Необычные файлы на компьютерах
- Физические носители информации

### Категория D: Следы присутствия

**Признаки:**
- Не объекты сами по себе, а следы
- Доказательства чьего-то присутствия
- Паттерны активности

**Примеры (гипотетические):**
- Необычные следы воздействия на материалы
- Паттерны движения (зафиксированные камерами)
- Остаточные энергетические сигнатуры (?)

## Процесс расследования

### Земная следственная группа

**Ресурсы (Месяц 1-6):**
- Сотни специалистов
- Временная база в зоне
- Защитные скафандры
- Сканирующее оборудование
- Лабораторный анализ

**Методы:**
1. **Полевые работы:**
   - Методичное прочесывание территории
   - Сканирование каждого метра
   - Фотографирование и документирование

2. **Лабораторный анализ:**
   - Химический анализ
   - Физические свойства
   - Проверка на радиацию и аномалии
   - Сравнение с базами данных

3. **Экспертная оценка:**
   - Консультации ученых
   - Технические эксперты
   - Специалисты разных профилей

**Результат:**
> "Титаническая работа в рекордные сроки."

---

### КД (Комитет Дизайнеров)

**Преимущества:**
- Полный доступ к данным земной группы
- Продвинутые технологии анализа
- Понимание протоколов физики
- Способность "видеть" изменения в константах

**Методы:**
1. **Реверс-инжиниринг:**
   - Анализ принципов работы
   - Восстановление протокола создания
   - Поиск технологической "подписи"

2. **Анализ следов:**
   - Остаточные изменения в физике
   - Точки доступа к протоколам
   - Энергетические паттерны

3. **Сравнение с базой данных:**
   - Тысячи предыдущих кейсов
   - Известные технологические "подписи"
   - Стили работы разных Дизайнеров

**Проблема:**
След обрывается на фейковых масках.

## Значение для сюжета

### Ключевая зацепка

**Потенциал:**
Среди 50 артефактов может быть ключ к раскрытию:
- Личности создателей поля
- Личности создателей урагана
- Мотивов обеих сторон

### Драматургическая функция

1. **Процесс расследования:**
   - Показывает масштаб усилий
   - Демонстрирует методичность
   - Создает напряжение (найдут ли что-то важное?)

2. **Точка пересечения:**
   - Земная группа и КД работают параллельно
   - Возможность сотрудничества
   - Конфликт интересов

3. **Ружье Чехова:**
   - Один из 50 артефактов может сыграть ключевую роль
   - Внезапное открытие
   - Поворот сюжета

- Координирует поиск артефактов

**Роль:**

Объясняет землянам, какие артефакты действительно важны (в рамках легенды).

### Ученые (Земля)

**Роль:**
- Технический анализ
- Лабораторные исследования

## Примечания для авторов

### Использование в сценах

**Варианты:**
1. **Процесс поиска:**
   - Показать методичность расследования
   - Рутинная работа + внезапная находка

2. **Анализ находок:**
   - Сцены в лаборатории
   - Дискуссии экспертов
   - КД объясняют (частично)

3. **Ключевое открытие:**
   - Один из артефактов дает зацепку
   - Поворотный момент расследования

### Баланс информации

- **Не перегружать:** Не нужно описывать все 50 артефактов
- **Фокус на ключевых:** 2-3 артефакта, важных для сюжета
- **Создать интригу:** Что это? Кто создал? Зачем?

### Тон

- **Научный подход:** Методичный анализ
- **Профессионализм:** Без паники, системная работа
- **Загадка:** Большинство артефактов остаются загадкой
