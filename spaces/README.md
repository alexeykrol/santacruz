# /spaces - Точки пространства

Эта папка содержит описание **точек пространства** - конкретных мест, где происходят сцены.

## Что такое "Точка пространства"

Точка пространства - это **конкретное место** внутри локации, где происходит сцена:
- Улица (угол 5-й авеню и 42-й улицы)
- Здание (конференц-зал отеля)
- Комната (кабинет следователя)
- Интерьер (кафе "Старбакс" на главной площади)

## Отличие от Локации

- **Локация** = широкое место (Нью-Йорк)
- **Точка пространства** = конкретное место (конференц-зал на Манхэттене)

## Структура

Точки пространства организованы по локациям:
```
/santa_cruz/          - Места в Санта-Круз
/san_francisco/       - Места в Сан-Франциско
/new_york/            - Места в Нью-Йорке
```

## Зачем нужны описания точек

Каждая точка пространства может использоваться в **нескольких сценах**. Описание точки содержит:
- Как выглядит место
- Какие объекты там есть
- Атмосфера
- Функциональное назначение

## Использование в сценах

В метаданных сцены указывается ссылка на точку пространства:
```markdown
---
space: /spaces/new_york/manhattan_conference_room.md
---
```
